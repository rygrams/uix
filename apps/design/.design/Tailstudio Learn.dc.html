Bienvenue dans la documentation Tailstudio
Tailstudio transforme votre design en classes Tailwind natives, en temps réel. Ces guides couvrent la prise en main, la gestion de votre design system, la collaboration et l'assistant IA.

Créer un projet
Depuis le tableau de bord, cliquez sur « Nouveau projet ». Choisissez un point de départ — page vierge ou modèle — puis ajoutez vos premiers frames depuis la barre d'outils.
Frames & breakpoints
Mobile, Tablet et Desktop sont liés : sélectionnez un calque comme le titre ou l'image et modifiez-le une fois pour qu'il se répercute sur les trois vues. Utilisez l'onglet Prototype pour relier vos écrans entre eux.
Styles & tokens
L'onglet Styles centralise vos styles de texte, couleurs et effets réutilisables — distincts du Thème global du projet. Toute modification s'applique partout où le style est utilisé.
Composants personnalisés
Sélectionnez un élément du canvas puis « Créer depuis la sélection » pour en faire un composant réutilisable, disponible ensuite dans le panneau Composants.
Thème du projet
Couleur d'accent, rayon des coins et police par défaut se règlent depuis l'onglet Thème et s'appliquent à l'ensemble du projet.
Commentaires
Chaque commentaire est ancré à un calque ou une page précise. Ouvrez le panneau Commentaires pour voir l'historique complet de chaque fil, répondre, ou marquer une discussion comme résolue.
Historique de versions
Nommez et enregistrez un instantané de votre thème, vos textes et vos composants à tout moment. Restaurez-le plus tard sans perdre votre travail en cours.
Partage en lecture seule
Depuis « Partager », activez le lien public pour laisser n'importe qui consulter une version figée de votre design, sans pouvoir la modifier.
Premiers pas avec l'assistant IA
Ouvrez l'assistant depuis le rail de gauche et décrivez la modification souhaitée en langage naturel — variante de composant, mise en responsive, audit d'accessibilité. Il applique le changement directement et vous montre le diff Tailwind.
Connecter le MCP
Le protocole MCP (Model Context Protocol) donne à Claude ou à tout agent compatible un accès direct à votre projet Tailstudio, pour continuer le travail côté code.
1. Ouvrir le panneau « Agent IA »
Dans l'éditeur, cliquez sur le bouton ✦ Agent IA en haut à droite.
2. Connecter le serveur MCP
Copiez la commande fournie et exécutez-la dans votre agent (Claude Code, Cursor, etc.) pour lier votre projet en lecture :
npx tailstudio-mcp connect
3. Partager le code d'un élément
Sélectionnez un calque, ouvrez le panneau « Agent IA » et cliquez sur « Copier le code pour l'agent » — collez ensuite le résultat directement dans la conversation de votre agent pour qu'il continue l'implémentation.
